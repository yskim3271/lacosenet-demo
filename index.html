DuBLoNet: Dual-Buffer Low-Latency Network
for Streaming Speech Enhancement

Submitted to Interspeech 2026
Abstract. Streaming speech enhancement requires balancing algorithmic latency against quality, yet existing approaches largely treat this as a binary causal versus non-causal choice. DuBLoNet addresses this with a single architecture through three mechanisms: (1) asymmetric temporal padding redistributes past and future context in convolutions, enabling systematic latency control without increasing model size; (2) dual-buffer streaming combines state buffers for past context with lookahead buffers for future context at input and feature levels; and (3) a context isolation mechanism prevents future-frame leakage into streaming state, ensuring training and inference consistency. On VoiceBank+DEMAND, DuBLoNet (1.37M parameters) covers 12.5 to 75.0 ms algorithmic latency, achieving PESQ from 3.35 at 12.5 ms to 3.43 at 75.0 ms. At just 12.5 ms (fully causal), DuBLoNet achieves PESQ 3.35, surpassing the prior causal state-of-the-art (3.27 at 46.5 ms).

Architecture

DuBLoNet is a frequency-domain causal speech enhancement model based on the Backbone encoder-decoder architecture. Asymmetric convolution padding ratios control the lookahead at each layer, enabling fine-grained latency configuration from a single trained model. At inference time, dual-buffer streaming processes audio frame-by-frame using state buffers (past context) and lookahead buffers (future context).
